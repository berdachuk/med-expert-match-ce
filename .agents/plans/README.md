# Plans

Milestone implementation plans. See root `AGENTS.md` → Plan Files for conventions.

## Naming

`M{NN}-{goal-slug}.md` — e.g., `M01-upgrade-spring-ai-to-m6.md`

- `{NN}` — zero-padded milestone number (01, 02, ..., 99)
- `{goal-slug}` — kebab-case short description
