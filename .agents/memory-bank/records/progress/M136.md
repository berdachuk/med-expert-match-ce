- **Milestone:** M136

## Summary

Migrated 2 structured-JSON LLM call sites from manual `.call().content()` + `ObjectMapper.readValue()` to `LenientJsonOutputConverter` (a `StructuredOutputConverter` implementation). Created `CaseAnalysisJson` and `GoalClassificationJson` records for typed parsing. Removed 80+ lines of manual parsing code (`parseCaseAnalysisResult`, `extractList`, `extractPotentialDiagnoses`, `extractJsonFromResponse`). Deprecated `LlmResponseSanitizer.extractJson()` in favor of the new converter. 11 new unit tests for `LenientJsonOutputConverter` covering fence-stripping, plain JSON, trailing prose, null/blank, nested braces, and domain record parsing.
