# DEC-004: External .st Prompt Templates

- **Status:** Accepted
- **Date:** 2026-06-13
- **Module:** llm
- **Affects:** `llm`, `core/config/PromptTemplateConfig`
- **Legacy alias:** D-004

## Rationale

No hardcoded prompt strings in Java; templates managed as resources under `src/main/resources/prompts/*.st` and wired via `PromptTemplate.builder().resource(...)`. Enables iteration without recompilation.

## Coupling note

Prompt short-key changes MUST update `core/util/LlmResponseSanitizer.java` (`FIELD_LABELS`, `JSON_BLOCK_PATTERN`) in lockstep — see `token-efficient-format` skill §sanitizer-coupling and `locks/README.md`.
