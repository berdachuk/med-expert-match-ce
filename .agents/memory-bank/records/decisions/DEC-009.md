# DEC-009: Graph Operations Through GraphService Only

- **Status:** Accepted
- **Date:** 2026-06-13
- **Module:** graph
- **Affects:** `graph`, `retrieval`, `llm`, `ingestion`
- **Legacy alias:** D-009

## Rationale

Centralized Cypher execution prevents injection, ensures connection management, single point for graph schema changes. All Cypher queries go through the `GraphService` interface; `MERGE` with all properties in-clause for idempotency; `REQUIRES_NEW` transaction isolation.
