# DEC-006: OpenAI-Compatible Providers Only

- **Status:** Accepted
- **Date:** 2026-06-13
- **Module:** core
- **Affects:** `core/config/SpringAIConfig`
- **Legacy alias:** D-006

## Rationale

MedGemma serves via OpenAI-compatible API; no Ollama/incompatible providers to minimize integration surface. Forbidden to change AI provider from OpenAI-compatible without explicit human approval.
