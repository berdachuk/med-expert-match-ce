# DEC-001: Java Records for Domain Entities

- **Status:** Accepted
- **Date:** 2026-06-13
- **Module:** all
- **Affects:** all domain modules
- **Legacy alias:** D-001

## Rationale

Immutable, concise data carriers; work well with JDBC RowMappers; reduce mutation bugs.

## Consequence

All domain entities (`Doctor`, `MedicalCase`, `Facility`, etc.) are Java records. DTOs and value objects follow the same pattern. Lombok is reserved for service impls only.
