# DEC-003: Separate Insert/Update Repository Methods

- **Status:** Accepted
- **Date:** 2026-06-13
- **Module:** all
- **Affects:** all repository modules
- **Legacy alias:** D-003

## Rationale

Explicit mutation intent; no `createOrUpdate`/upsert — forces thinking about writes. Each repository exposes distinct `insert()` and `update()` methods.
