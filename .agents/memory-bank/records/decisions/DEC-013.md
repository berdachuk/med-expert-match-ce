# DEC-013: Array-Based References (TEXT[]) Over Foreign Keys

- **Status:** Accepted
- **Date:** 2026-06-13
- **Module:** all
- **Affects:** `doctor`, `medicalcase`, `medicalcoding`, `facility`
- **Legacy alias:** D-013
- **Reference:** `docs/pipeline/02-architecture.md` (Array-Based References Pattern)

## Rationale

Read-heavy workload optimization: GIN indexes for fast containment queries (`@>`), no JOIN overhead, simple application code. Trade-off: no DB-enforced referential integrity. Graph layer handles semantic relationships; ingestion validates reference data.
