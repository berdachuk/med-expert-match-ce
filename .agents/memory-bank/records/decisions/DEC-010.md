# DEC-010: Mock External Services in ITs (M52)

- **Status:** Accepted
- **Date:** 2026-06
- **Module:** evidence
- **Affects:** `evidence`, `retrieval`, `llm`
- **Legacy alias:** D-010

## Rationale

WireMock fixtures for all external HTTP APIs; no live calls from tests. Record real responses once, store in `src/test/resources/{module}/`. External services that accept an injectable base URL enable easy WireMock wiring.
