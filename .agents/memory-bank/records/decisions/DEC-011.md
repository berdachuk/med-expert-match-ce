# DEC-011: AutoMemory — Explicit Tools Over Advisor (M15)

- **Status:** Accepted
- **Date:** 2026-06
- **Module:** llm
- **Affects:** `llm/automemory/`
- **Legacy alias:** D-011
- **Reference:** `docs/decisions/M15-automemory-advisor-decision.md`

## Rationale

Kept explicit `AutoMemoryTools` + memory system prompt (Option B) instead of migrating to `AutoMemoryToolsAdvisor` (Option A). Explicit tools give the agent more control over what/when to remember. The current system is stable and tested.
